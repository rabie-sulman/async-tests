
# async-tests

## install dependencies:

### php

```console
> cd php_example
> composer install
```

### node

```console
> cd node_example
> npm install
```
  

## test

### node

```console
> node node_example/encode.js
> node node_example/verify.js
```

### php

```console
> php php_example/encode.php
> php php_example/verify.php
```
